🏗️ 系統架構說明

System Architecture Overview

🛠️ 核心技術棧 (Tech Stack)

FastAPI 高效能 Python 非同步框架，負責處理所有 API 請求與路由分發。
SQLAlchemy 工業級 Python ORM，負責與資料庫進行物件對映與安全查詢。
Uvicorn 基於 uvloop 的 ASGI 伺服器，支撐極低延遲的連線回應。
Jinja2 強大的模板引擎，負責後端渲染動態 HTML 頁面。

🔒 安全與認證機制

  • 密碼加密： 採用 bcrypt 演算法進行強力雜湊運算，確保資料庫遭洩漏時原始密碼不外流。
  • JWT 身份驗證： 使用 JSON Web Tokens 簽發令牌，有效期限設定為 24 小時。
  • Cookie 安全傳輸： 令牌儲存於 HttpOnly 屬性的 Cookies 中，有效防範 XSS (跨站腳本攻擊) 竊取 Session。
  • 角色型權限控制 (RBAC)： 系統嚴格區分 Reporter 與 Technician 權限。非技術員無法存取狀態修改與維修日誌寫入 API。

🔄 請求處理生命週期

1. 用戶請求 (Request)： 瀏覽器發送 HTTP 請求。
2. 依賴注入 (Dependency Injection)： FastAPI 自動獲取資料庫連線並驗證 JWT Cookies。
3. 業務邏輯 (Controller)： `main.py` 處理報修邏輯或更新日誌。
4. 模板渲染 (Rendering)： Jinja2 將處理後的資料填入 HTML 模板。
5. 回應 (Response)： 回傳完整的 HTML 頁面或執行跳轉 (Redirect)。
