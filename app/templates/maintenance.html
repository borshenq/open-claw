⚙️ 系統維護操作說明

System Maintenance & Operations Guide

🚀 服務啟動與管理

本系統使用 Uvicorn 作為 ASGI 伺服器，建議在虛擬環境 (venv) 中執行。

# 進入虛擬環境並啟動服務 (開發模式)
source venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
# 背景執行並記錄日誌 (生產模式)
nohup python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000 > server.log 2>&1 &

💾 資料庫維護與重置

系統使用 SQLite 檔案型資料庫 sql_app.db。維護時請注意以下操作：

⚠️ 注意： 刪除資料庫檔案將導致所有報修資料、用戶資料永久遺失，請務必先進行備份。
  • 備份： 直接複製 sql_app.db 檔案至安全位置即可。
  • 重置並重新生成測試資料： 若需清空資料並恢復初始測試狀態，請執行：
rm sql_app.db
python3 seed_data.py

📁 檔案與上傳管理

所有報修上傳的照片均儲存於靜態目錄中：

  • 儲存路徑： app/static/uploads/
  • 自動命名： 系統會自動將檔案重新命名為 UUID (隨機編碼)，以防檔名衝突。
  • 空間維護： 若空間不足，管理員可手動清理該資料夾下的舊照片。

📦 套件與依賴更新

當系統需要更新或搬移環境時，請確保依賴項完整：

# 更新依賴清單
pip install -r requirements.txt
