⚙️ 基地維護小筆記

身為基地的守護者，這些操作秘訣一定要記下來喔！✨

🚀 讓基地動起來

就像啟動飛船一樣，我們需要透過指令讓基地開始運作！

# 喚醒虛擬環境 (準備出發) source venv/bin/activate

# 啟動基地引擎 (開發模式) uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
# 讓基地進入自動導航 (生產模式) nohup python3 -m uvicorn app.main:app --host 0.0.0.0 --port 8000 > server.log 2>&1 &

💾 保管好資料保險箱

資料庫 sql_app.db 是我們最重要的寶物，要小心照顧！

超級注意： 如果弄丟了資料庫檔案，所有小隊員的報修紀錄都會消失！請一定要先影印備份喔！
  • 影印備份： 直接把 sql_app.db 複製一份到安全的地方就好囉。
  • 重新打掃基地： 如果想清空所有資料重新開始，請執行：
# 搬走舊箱子，裝進新資料 rm sql_app.db
python3 seed_data.py

📁 整理照片小抽屜

小隊員們拍的照片都放在這個神秘資料夾裡：

  • 抽屜位置： app/static/uploads/
  • 自動變身： 系統會給照片取一個像 UUID 的怪名字，這樣就不會搞混了。
  • 大掃除： 如果抽屜太擠了，可以手動把裡面的舊照片清掉喔！🧹

📦 更新工具包

如果基地需要更多新功能，記得更新你的工具包：

# 安裝所有必備小零件 pip install -r requirements.txt
