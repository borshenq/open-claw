📊 資料庫結構說明

Database Schema & ER Model

👥 Users (用戶資料表)

存取系統所有角色的基本資料與認證資訊。

欄位名類型約束說明
idIntegerPK自增主鍵
usernameString(50)Unique, Index唯一登入名稱
emailString(100)Unique, Index聯絡信箱
hashed_passwordString(255)NotNull密碼雜湊
roleString(20)Default: "Reporter"角色身份：Reporter, Technician

🛠️ Repair Requests (報修申請表)

核心業務表，記錄全校所有報修項目。

欄位名類型約束說明
idIntegerPK自增主鍵
item_nameString(100)NotNull損壞設備名稱
locationString(100)NotNull損壞位置
descriptionTextNotNull詳細問題描述
image_urlString(255)Nullable上傳照片之路徑
statusString(20)Default: "Pending"目前狀態：Pending, In Progress, Completed
created_atDateTimeAutoNow申請建立時間
reporter_idIntegerFK關聯到 Users.id

📝 Repair Logs (維修日誌表)

追蹤單一報修項目的多次更新紀錄。

欄位名類型約束說明
idIntegerPK自增主鍵
request_idIntegerFK關聯到 RepairRequests.id
noteTextNotNull維修動作或進度說明
updated_atDateTimeAutoNow日誌建立時間
